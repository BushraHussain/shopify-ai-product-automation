# AI-Powered Shopify Product Upload Automation

## Overview

This project automates Shopify product creation using n8n and OpenAI.

The workflow:

1. Receives product data
2. Generates SEO-friendly product descriptions using AI
3. Formats product information
4. Uploads products directly to Shopify
5. Returns confirmation and product details

## Business Problem

Manual product uploads required repetitive copywriting and data entry.

This automation reduced manual effort, improved consistency, and accelerated product onboarding.

## Workflow Architecture

Product Data
↓
OpenAI
↓
Description Generation
↓
Formatting
↓
Shopify API
↓
Product Creation

## Technologies

- n8n
- OpenAI API
- Shopify API
- REST APIs
- Workflow Automation
